Objective :
		To read the contents of the file which contains songs and authors separated by forward slash
		Parse the content , and retrieve only the song title and put them into ArrayList
		And sort the list obtained in ascending order
		
Approach :
		Create a file and place that in the project directory
		Read the file using Java File Handling concepts
		Using the Collection's ArrayList class , store the list of songs that is parsed
		Inorder to sort the list , we can make use of the inbuilt methods provided by the Collections class
